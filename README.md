# curso-microservicios
Implementacion de microservicios usando Cliente REST de Feign y RestTemplate para el consumo de APIs.
* Balanceo de carga del lado del cliente con Ribbon.
* Creacion de un servidor de registro con Eureka.
* Manejo de tolerancia de fallos y latencia con Hystric.
* Configuracion de servidor Zuul como API Gateway.
* Configuracion de Profiles.
